== Description ==

The Sitemaps plugin will generate an XML sitemap of your WordPress website and ping the most popular search engines: Google, Bing, and Ask.com.

== Installation ==

1. Upload the folder to `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to " Xml Sitemap Settings" and build your Sitemap.
